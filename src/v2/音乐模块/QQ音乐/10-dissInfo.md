# 获取歌单的歌曲列表API

## 简要描述

- 获取账号上的收藏歌曲，自建歌单等信息，需配合歌单列表使用。
- 所需参数在搜索或获取播放地址时获取

## 请求URL
- `https://api.vkeys.cn/v2/music/tencent/dissinfo`

## 请求方式
- `GET` / `POST`

## 请求示例
- https://api.vkeys.cn/v2/music/tencent/dissinfo?id=1754192231

## 请求参数

| 参数名 |是否必选|数据类型|说明   |
|:----    |:---|:----- | :-------|
| id   | 是   | int    | 歌单id    |
| page | 否   | int    | 页数，默认为1    |
| num  | 否   | int    | 每页显示数，默认为10    |
